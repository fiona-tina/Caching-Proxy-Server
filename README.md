**Language**: C++  

**Developers**:

## Usage:
* Open folder "server"
* Run the "make" command
* Run ./proxy_daemon
* Change your browser's proxy settings to the IP of the system he program is running on.

## Disclaimer:
* Code is buggy and may not be a fully functional proxy.
* Supports only GET, POST and CONNECT.
